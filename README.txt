AI Fitness Planner (Fully Local)
================================

How to run
----------
1) Open "index.html" in a browser (double-click).
2) Go to "Profile" and fill in your goal + details.
3) Click "Save profile", then "Generate 7-day plans".
4) Use the nav bar to switch between:
   - Workouts (7-day workout schedule, highlights urgent/missed days)
   - Meals (7-day meal plan, adapts portions to calorie target)
   - Tracker (log workouts done, weight, measurements)
   - Calculator (BMI, BMR, TDEE, calorie + macro targets)

Offline + privacy
-----------------
- No backend and no API calls.
- Your data is stored only in this browser using localStorage.
- To reset everything: Profile -> "Clear all local data".
